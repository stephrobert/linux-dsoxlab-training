## What this lab is about

Before working on a system you need to know exactly what you are dealing with.
This lab teaches you to retrieve four key pieces of information about any Linux machine:
hostname, distribution, kernel version and IP address.

These are the first four checks of any new server on-boarding checklist.

## What you will do

1. Find the machine hostname
2. Find the distribution name and version
3. Find the running kernel version
4. Find the local IP address
5. Write all four to `vm-info.txt`
