# Lab l1-04 — First steps in the terminal

**Level**: L1 — Fundamentals (B0) | **Runtime**: shell | **Time**: 15 min

## Goal

Run four essential commands and capture their output in a text file.
This exercise builds the reflex of identifying yourself and your environment
as soon as you open a terminal session.

## Commands practiced

| Command | Output |
|---------|--------|
| `whoami` | Current username |
| `hostname` | Machine name |
| `echo $HOME` | Home directory path |
| `date` | Current date and time |
| `echo "text" > file` | Write text to a file (overwrite) |
| `echo "text" >> file` | Append text to a file |
