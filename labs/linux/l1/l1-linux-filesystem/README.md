# Lab l1-07 — Linux Filesystem Hierarchy (FHS)

**Level**: L1 — Fundamentals (B1) | **Runtime**: shell | **Time**: 20 min

## Goal

Understand the Filesystem Hierarchy Standard (FHS) — where things live on a Linux system
and why. By mastering this map you will instantly know where to look for logs, configs,
binaries, and device files on any distribution.
