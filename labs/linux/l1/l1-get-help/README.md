# Lab l1-06 — Get help from the command line

**Level**: L1 — Fundamentals (B0) — Checkpoint | **Runtime**: shell | **Time**: 20 min

## Goal

Learn to use the three built-in help tools every Linux administrator relies on daily:
`man`, `--help`, and `apropos`. By the end of this lab you will never need a search
engine to find a command option.
