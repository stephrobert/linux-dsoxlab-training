# Lab l1-05 — Read and decode a command

**Level**: L1 — Fundamentals (B0) | **Runtime**: shell | **Time**: 20 min

## Goal

Learn to decompose any Linux command into three parts:
**command**, **options**, **arguments**.
Then practice identifying and fixing incorrect commands.

## Anatomy of a command

```
ls  -la  /etc
│    │    └── argument (what to act on)
│    └──────── options  (how to do it)
└───────────── command  (what to run)
```
