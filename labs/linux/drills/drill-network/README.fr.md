# Drill — réseau statique

> Préparer : `dsoxlab provision` puis `dsoxlab run drill-network`

**4 tâches, 100 points, 20 minutes. Aucun indice.** Partagé entre RHCSA et LFCS —
le sujet ne nomme aucun outil. Tout sur `lab0`, **jamais** l'interface de
gestion. Vérifié **après un rechargement du réseau**.

```bash
dsoxlab challenge drill-network        # le sujet
dsoxlab check drill-network            # se faire noter (AlmaLinux — nmcli)
dsoxlab check drill-network -t ubuntu  # se faire noter (Ubuntu — netplan)
```
