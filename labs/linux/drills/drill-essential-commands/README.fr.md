# Drill — commandes essentielles

> Préparer : `dsoxlab provision` puis `dsoxlab run drill-essential-commands`

**5 tâches, 100 points, 20 minutes. Aucun indice.** Partagé entre RHCSA et LFCS —
ces commandes sont identiques sur les deux distributions.

## Le passer

```bash
dsoxlab challenge drill-essential-commands        # le sujet
dsoxlab check drill-essential-commands            # se faire noter (AlmaLinux)
dsoxlab check drill-essential-commands -t ubuntu  # se faire noter (Ubuntu)
```
