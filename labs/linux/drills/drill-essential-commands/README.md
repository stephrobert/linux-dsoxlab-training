# Drill — essential commands

> Prepare: `dsoxlab provision` then `dsoxlab run drill-essential-commands`

**5 tasks, 100 points, 20 minutes. No hints.** Shared between RHCSA and LFCS —
these commands are identical on both distributions.

## Take it

```bash
dsoxlab challenge drill-essential-commands        # the subject
dsoxlab check drill-essential-commands            # score it (AlmaLinux)
dsoxlab check drill-essential-commands -t ubuntu  # score it (Ubuntu)
```
