# Drill — pare-feu

> Préparer : `dsoxlab provision` puis `dsoxlab run drill-firewall`

**5 tâches, 100 points, 20 minutes. Aucun indice.** Partagé entre RHCSA et LFCS —
le sujet ne nomme aucun outil. Les règles sont vérifiées **après un rechargement**.

```bash
dsoxlab challenge drill-firewall        # le sujet
dsoxlab check drill-firewall            # se faire noter (AlmaLinux — firewalld)
dsoxlab check drill-firewall -t ubuntu  # se faire noter (Ubuntu — ufw)
```
