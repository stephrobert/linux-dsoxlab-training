# Drill — package management

> Prepare: `dsoxlab provision` then `dsoxlab run drill-packages`

**5 tasks, 100 points, 20 minutes. No hints.** Shared between RHCSA and LFCS —
the subject names no tool: use your distribution's.

```bash
dsoxlab challenge drill-packages        # the subject
dsoxlab check drill-packages            # score it (AlmaLinux — dnf)
dsoxlab check drill-packages -t ubuntu  # score it (Ubuntu — apt)
```
