# Drill — gestion des paquets

> Préparer : `dsoxlab provision` puis `dsoxlab run drill-packages`

**5 tâches, 100 points, 20 minutes. Aucun indice.** Partagé entre RHCSA et LFCS —
le sujet ne nomme aucun outil : emploie celui de ta distribution.

```bash
dsoxlab challenge drill-packages        # le sujet
dsoxlab check drill-packages            # se faire noter (AlmaLinux — dnf)
dsoxlab check drill-packages -t ubuntu  # se faire noter (Ubuntu — apt)
```
