# Capstone — LFCS mock exam

> Prepare: `dsoxlab provision` then `dsoxlab run lfcs-mock-exam`

[**LFCS on the companion guide**](https://blog.stephane-robert.info/docs/admin-serveurs/linux/certifications/lfcs/)

**17 tasks, 100 points, 1 Ubuntu VM, 120 minutes. 70/100 to pass. No hints.**

The 5 official domains at their real weights: Essential Commands (20%),
Operations Deployment (25%), Users and Groups (10%), Networking (25%),
Storage (20%).

Everything must be **persistent after reboot** — that is what the tests check.

## Take the exam

```bash
dsoxlab challenge lfcs-mock-exam   # the subject: 17 tasks
dsoxlab check lfcs-mock-exam       # score it
```
