# Capstone — examen blanc LFCS

> Préparer : `dsoxlab provision` puis `dsoxlab run lfcs-mock-exam`

[**Le LFCS sur le guide compagnon**](https://blog.stephane-robert.info/docs/admin-serveurs/linux/certifications/lfcs/)

**17 tâches, 100 points, 1 VM Ubuntu, 120 minutes. 70/100 pour réussir. Aucun
indice.**

Les 5 domaines officiels à leurs poids réels : Essential Commands (20 %),
Operations Deployment (25 %), Users and Groups (10 %), Networking (25 %),
Storage (20 %).

Tout doit être **persistant après reboot** — c'est ce que les tests vérifient.

## Passer l'examen

```bash
dsoxlab challenge lfcs-mock-exam   # le sujet : 17 tâches
dsoxlab check lfcs-mock-exam       # se faire noter
```
